== Description ==

Tabtastic QuickTabs makes creating and displaying tabbed content on your website a breeze. QuickTabs are specifically designed to use the DMS Editor to create multicolored tabs as design elements

## Features

* Individual color controls for each tab
* Create multicolored tab sets
* Multiple QuickTabs on a page
